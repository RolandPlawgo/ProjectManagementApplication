# Scrum Project Management Web Application
A thesis project built with ASP.NET Core to support teams in applying Scrum principles.


# Overview
## Purpose
This web application was developed as my bachelorâ€™s thesis in Computer Science.
## Description
The application enables Scrum teams to manage backlog, plan sprints, organize tasks on a Scrum board, and schedule meetings. It supports all key Scrum roles, artifacts, and events.

## Technologies used
- C#
- ASP .NET Core (MVC)
- Entity Framework Core
- JavaScript
- HTML, CSS, SCSS
- Bootstrap


# Features

**Scrum master**
- Managing projects
- Managing users
- Adding and managing meetings
- Starting sprints
  
**Product owner**
- Adding user stories to the backlog
- Ending sprints (and accepting completed tasks)

**Developer (default role)**
- Adding user stories to sprints during sprint planning
- Adding subtasks to user stories during sprint planning
- Picking tasks to complete during sprints
- Marking tasks on the scrum board  as done
- Adding comments to tasks on the scrum board
